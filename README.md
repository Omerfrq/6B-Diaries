# 6B-Diaries
A Simple web diary to learn about BSIT-6B Students

To install the Dependencies run
```
npm install
```
To Run the Project
```
npm start
```
And view the Website at
[http://localhost:8080/](http://localhost:8080/)

To compile change in the Project
```
npm run dev
```
To build the Project 
```
npm run dev
```
